🛒 E-Commerce Sales Analysis

📌 Project Title: E-Commerce Sales Analysis Dashboard

🎯 Objective: To analyze business sales performance and identify trends across regions, product categories, and discount impact using data visualization

📊 KPIs Used:
Total Sales
Total Profit
Profit Margin %
Total Orders

📈 Insights Generated:
Technology category contributes highest sales
High discounts negatively impact profit
Some regions outperform others in sales

📊 Dashboard Features:
Region-wise Sales Analysis
Category-wise Performance
Profit vs Discount Relationship
Interactive Filters

🛠 Tools Used:
Power BI
DAX
Data Visualization
